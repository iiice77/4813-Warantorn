"คำสั่งพื้นฐาน HTML"
ลำดับที่ คำสั่ง ความหมาย ตัวอย่าง
1 < html > เริ่มต้นและสิ้นสุดเอกสาร < html >...< /html >
2 < head > กำหนดชื่อเรื่อง < head >...< /head >
3 < body > เริ่มต้นและสิ้นสุดเนื้อหาบนหน้าเว็บ < body >...< /body >
4 < title > ชื่อเรื่องที่ปรากฏบน title bar < title >...< /title >
5 < h1 > to < h6 > กำหนดขนาดตัวอักษร < h4 >...< /h4 >
6 < p> กำหนดย่อหน้าใหม่ < p >...< /p >
7 < hr > ขีดเส้นใต้เมื่อจบบรรทัด < hr >...< /hr >
8 < br > ขึ้นบรรทัดใหม่ < br >...< /br >
9 size=""noshade"" กำหนดความหนาของเส้นใต้ < hr size="" noshade "" >
10 < !--ข้อความ-- > คำสั่งหมายเหตุ < !--ข้อความ-- >
11 & nbsp ; เพิ่มช่องว่าง & nbsp ;
12 < center > จัดให้ข้อความอยู่กึ่งกลาง < center >...< /center >
13 < b > ตัวอักษรหนา < b >...< /b >
14 < i > ตัวอักษรเอียง < i >...< /i >
15 < u > ขีดเส้นใต้ตัวอักษร < u >...< /u >
16 < tt > ตัวอักษรแบบพิมพ์ดีด < tt >...< /tt >
17 < font color="red" > กำหนดสีตัวอักษร < font color="red" >...< /font >
18 < font size = "4" > กำหนดขนาดตัวอักษร < font size = "4" >...< /font >
19 < font face = "Arial" > กำหนดรูปแบบตัวอักษร < font face ="Arial" >...< /font >
20 < besefont size = "2" > กำหนดค่าเริ่มต้นของขนาดตัวอักษร < besefont size ="2" >...< /besefont >
21 < table > สร้างตาราง < table >< /table >
22 < meta > แสดงข่าวสารของไฟล์ HTML < meta >
23 < background > ใช้ร่วมกับ < body >หรือ < table >เพื่อกำหนดพื้นหลังเป็นรูปภาพ < body background="" >
24 < bgcolor > ใช้ร่วมกับ < body >หรือ< table >เพื่อกำหนดพื้นหลังเป็นสี < bgcolor="blue" >
25 < ul > แสดงรายการโดยไม่เรียงลำดับ < ul >...< /ul >
26 < ol > แสดงรายการโดยเรียงลำดับ < ol >...< /ol >
27 < li > แสดงข้อความในแต่ละบรรทัดของ < ol > หรือ < ul > < li >...< /li >
28 < a > กำหนดการเชื่อมโยง < a >...< /a >
29 < img > กำหนดการเชื่อมโยงด้วยรูป < img >...< /img >
30 < tr > กำหนดแถวในตาราง < tr >...< /tr >
31 < td > กำหนดคอลัมน์ในแถว < td >...< /td >
32 rowspan รวมแถว < td rowspan="จำนวนแถว" >
33 colspan รวมสดมณ์ < td colspan="จำนวนเซลล์">
34 < th > กำหนดหัวข้อเรื่อง < th >...< /th >
35 < form > กำหนดสร้างฟอร์ม < form >...< /form >
36 < input > กำหนด input ใน form < input type="text" >
